# vscode-directx-playground

VSCode + MSVC で C++ / DirectX を学習するためのリポジトリ。

## Requirements
- Windows 10 / 11
- Visual Studio Build Tools
- Visual Studio Code

tasks.jsonに記述しているVsDevCmd.batのパスは各自の環境に合わせて変更してください。

## Build
Open this project from **Developer Command Prompt for VS**, then:

- `Ctrl + Shift + B` : Build
- `F5` : Run / Debug
